# ShopEase E-Commerce Site

A professional, responsive e-commerce website with dashboard functionality built using HTML, CSS, JavaScript, and Bootstrap.

## Features

- **25 Products**: Across 5 categories (Electronics, Home & Kitchen, Sports & Outdoors, Fashion)
- **Responsive Design**: Fully responsive on all screen sizes using Bootstrap
- **Animations**: Smooth animations throughout the site
- **Shopping Cart**: Add/remove items, view cart modal
- **Checkout Process**: Complete checkout form with validation
- **Dashboard**: View order statistics and order history
- **Local Storage**: Persistent cart and order data

## Files Structure

- `index.html` - Main homepage with product listings
- `dashboard.html` - Admin dashboard showing orders and statistics
- `checkout.html` - Checkout page with billing form
- `styles.css` - Custom CSS with animations and responsive styles
- `script.js` - Main JavaScript for product display and cart functionality
- `dashboard.js` - JavaScript for dashboard statistics and order display
- `checkout.js` - JavaScript for checkout process

## How to Use

1. Open `index.html` in a web browser
2. Browse products and add items to cart
3. Click on cart icon to view cart
4. Click "Checkout" to proceed to checkout
5. Fill in billing information and place order
6. View orders in the dashboard

## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript (ES6+)
- Bootstrap 5.3.0
- Font Awesome icons
- Local Storage for data persistence

## Responsive Breakpoints

- Desktop: > 1200px
- Tablet: 768px - 1200px
- Mobile: < 768px

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Development

To run locally, serve the files using a local web server to avoid CORS issues with external resources.
